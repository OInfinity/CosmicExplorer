## How to Use

1. Open `LearnMore.html` in any modern browser  
2. Scroll through discoveries using the left panel  
3. The right panel updates visually as chapters change  
4. Navigation buttons offer quick switching and page transitions  

No setup. No dependencies. Pure front-end execution.
